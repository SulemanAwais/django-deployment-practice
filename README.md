## 📦 Project Setup

### Requirements

- Python 3.10
- Django 4.0.6

### Installation

1. **Clone the Repository**

   ```bash
   git clone https://github.com/SulemanAwais/django-deployment-practice.git
   ```
2. **Navigate to the Project Directory**
   ```bash
   cd django-deployment-practice
  ```
3. **Create a Virtual Environment**
